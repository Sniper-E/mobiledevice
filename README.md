# Mobile Device

## Discription ---
Adds [ Post made via Mobile Device ] when posting from a mobile device. Welcome mobile user message on index page showing user's mobile device. Displays logged in user's mobile image in header navbar on all other pages. Displays user is on their mobile in topic mini-profile and profile page. Popup mobile logs to view any user on a mobile device who has visited your site. Selections in UCP / Board preferences / Edit display for user options. Selections in ACP / Extensions / Mobile device / Settings for enabling features. Permissions / Misc for selecting who can view logs and who can clear logs.

## Install ---
1. In the `root/ext/` directory of your phpBB board, create a new directory named `sniper`
2. Copy the `mobiledevice/` folder into `root/ext/sniper/`
3. Navigate in the ACP to `Customise -> Manage extensions`
4. Look for `Mobile Device` under the Disabled Extensions list and click `Enable`

## Update ---
1. Navigate in the ACP to `Customise -> Extension Management -> Extensions`
2. Look for `Mobile Device` under the Enabled Extensions list and click `Disable`
3. Copy the `mobiledevice/` folder into `root/ext/sniper/` over write existing files
4. Navigate in the ACP to `Customise -> Manage extensions`
5. Look for `Mobile Device` under the Disabled Extensions list and click `Enable`

## Uninstall ---
1. Navigate in the ACP to `Customise -> Extension Management -> Extensions`
2. Look for `Mobile Device` under the Enabled Extensions list and click `Disable`
3. To permanently uninstall, click `Delete Data` and then delete the folder `/ext/sniper/mobiledevice`
